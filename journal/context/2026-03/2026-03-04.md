Dual-export not feasible since managed settings lock all endpoint vars.

## Key learnings
1. Claude Code events use OTEL logs protocol, not metrics — need OTEL_LOGS_EXPORTER
2. Managed settings (MDM) have highest precedence in Claude Code — cannot be overridden
3. Official Datadog MCP server is a remote HTTP endpoint with OAuth, not a local npm package
4. Service name "claude-code" is hardcoded in resource attributes
5. Delta temporality is the default (Datadog-compatible, no override needed)
