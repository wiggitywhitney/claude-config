All 92 Python tests pass with identical results to bash originals.

### Technical Decisions
- TestResults class uses `assert_allow`/`assert_deny` for hook tests and `assert_field`/`assert_tier` for script JSON output tests — matching the two distinct bash test patterns
- `run_hook()` pipes JSON via stdin to subprocess, `run_script()` passes directory as positional arg with env control
- TempDir context manager handles fixture cleanup; write_file() creates parent dirs automatically
- Combined runner completes all 3 files in ~19.9s; commit-message alone ~6.8s vs bash ~20s (~66% faster)
