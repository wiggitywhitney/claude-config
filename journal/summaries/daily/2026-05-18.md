## Key Decisions

- **Add Read, Write, and Edit tool permissions to /make-autonomous** — Child sessions need these to create files, update PRD checkboxes, and write pause markers. Without them, every session fails immediately. Discovered during validation when the first attempt couldn't create implementation files.

- **Fix count_milestone_iterations arithmetic crash from grep -c exit code** — macOS grep outputs the count but exits with code 1 when no matches are found. The previous `|| echo 0` fallback duplicated the zero, breaking arithmetic. Changed to `|| true` to suppress the exit code without adding extra output.

- **Design iteration cap smoke test scenarios to resist standard bats mocking patterns** — Using an unreachable domain alone isn't enough; children can make tests pass through proper test engineering (injecting fake curl on PATH). Future iteration cap tests need success criteria that can't be satisfied by implementing the script differently.

- **Add M7 real PRD execution validation milestone before M8 documentation** — M6 validated orchestrator mechanics with synthetic scenarios. Only running against a real PRD in a different repo proves cross-repo portability. M8 documentation now depends on M7 completing and includes observations from that run.

- **Target spinybacked-orbweaver PRD #752 for M7 real PRD validation instead of Journal PRD #46** — PRD #46 is low priority in Journal's queue. PRD #752 has real TypeScript code, an actual test suite, docs generation, and a natural human pause point. This exercises genuine TDD and human-in-the-loop handling on real work rather than contrived scenarios.
